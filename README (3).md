# Параметризация малой молекулы полем OpenFF

Скрипт предназначен для генерации (`.gro`, `.top`) файлов Gromacs'a для лигандов из входного файла `.sdf` с использованием OpenFF toolkit

# Основные возможности

Скрипт реализует следующий пайплайн:

```
SDF -> OpenFF Molecule
    -> генерация 3D-конформера (если координаты отсутствуют)
    -> метод расчёта частичных зарядов  (AM1-BCC)
    -> параметризация с помощью силового поля SMIRNOFF 
    -> экспорт топологии в формате GROMACS
```

Поддерживается обработка нескольких молекул из одного SDF-файла.

Функциональность:

* Автоматическое извлечение имени молекулы из SDF
* Создание `.gro`, `.top`, `.mdp`
* Генерация 3D-конформера 
* Подсчет зарядова
* Опция оценки оставшегося времени выполнения (ETA)
* Опция quiet-mode для подавления предупреждений

# Требования

Рекомендуемая среда:

Python 3.10

Linux (рекомендуется Ubuntu 22.04 или совместимый дистрибутив). Также поддерживается запуск через WSL с Ubuntu 22.04 на Windows (задание было выполнено именно в такой конфигурации).

Необходимые библиотеки:

```
openff-toolkit
openff-interchange
rdkit
ambertools
openmm
```

Перед установкой необходимо убедиться, что на системе установлена Conda (например, Miniconda или Anaconda). После установки рекомендуется создать отдельное виртуальное окружение для проекта и установить в него все необходимые зависимости одной командой:

```
conda create -n openff_env python=3.10
conda activate openff_env

conda install -c conda-forge \
openff-toolkit \
openff-interchange \
rdkit \
ambertools \
openmm
```

# Использование

Базовый запуск:

```
python main.py input.sdf output_directory
```

Пример:

```
python main.py example.sdf output
```

# CLI-параметры

Скрипт поддерживает следующие дополнительные параметры командной строки:

## --quiet

Подавляет предупреждения и сообщения сторонних инструментов, перенаправляя stderr в devnull.

Пример использования:

```
python main.py example.sdf output --quiet
```

## --no-progress

Отключает отображение оценки оставшегося времени выполнения (ETA).

Пример использования:

```
python main.py example.sdf output --no-progress
```

# Выходные файлы

Для каждой молекулы генерируются:

```
MOLNAME.gro
MOLNAME.top
MOLNAME_pointenergy.mdp
```

Имя молекулы автоматически извлекается из первой строки блока SDF.

# Основные параметры

* Силовое поле:
  openff_unconstrained-2.1.0.offxml

* Метод расчёта частичных зарядов:
  AM1-BCC

* Размер  ячейки для GROMACS:
  4 × 4 × 4 нм
